# az network virtual-network-appliance

```bash
# Create a virtual network appliance.
az network virtual-network-appliance create

# Delete the specified virtual network appliance.
az network virtual-network-appliance delete

# List all virtual network appliances in a subscription.
az network virtual-network-appliance list

# Get information about the specified virtual network appliance.
az network virtual-network-appliance show

# Update a virtual network appliance.
az network virtual-network-appliance update

# Place the CLI in a waiting state until a condition is met.
az network virtual-network-appliance wait
```
