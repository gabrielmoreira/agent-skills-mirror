# az logicapp

```bash
# Create a logic app.
az logicapp create

# Delete a logic app.
az logicapp delete

# List logic apps.
az logicapp list

# Restart a logic app.
az logicapp restart

# Scale a logic app.
az logicapp scale

# Get the details of a logic app.
az logicapp show

# Start a logic app.
az logicapp start

# Stop a logic app.
az logicapp stop

# Update a logic app.
az logicapp update

# Delete a logic app's settings.
az logicapp config appsettings delete

# Show settings for a logic app.
az logicapp config appsettings list

# Update a logic app's settings.
az logicapp config appsettings set

# Perform deployment using the kudu zip push deployment for a logic app.
az logicapp deployment source config-zip
```
