# az mdp

```bash
# Create a pool.
az mdp pool create

# Delete a pool.
az mdp pool delete

# List all pools.
az mdp pool list

# Get a pool.
az mdp pool show

# Update a pool.
az mdp pool update

# Place the CLI in a waiting state until a condition is met.
az mdp pool wait

# List resource agents by Pool.
az mdp pool agent list

# List sku resources in given location.
az mdp sku list

# List quota resources in given location.
az mdp usage list
```
