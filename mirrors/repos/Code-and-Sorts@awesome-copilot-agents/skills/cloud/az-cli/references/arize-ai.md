# az arize-ai

```bash
# Create a OrganizationResource.
az arize-ai observability-eval organization create

# Delete a OrganizationResource.
az arize-ai observability-eval organization delete

# List OrganizationResource resources by subscription ID.
az arize-ai observability-eval organization list

# Get a OrganizationResource.
az arize-ai observability-eval organization show

# Update a OrganizationResource.
az arize-ai observability-eval organization update

# Place the CLI in a waiting state until a condition is met.
az arize-ai observability-eval organization wait
```
