# az cloudhsm

```bash
# Create a Cloud HSM.
az cloudhsm create

# Delete a Cloud HSM.
az cloudhsm delete

# List Cloud HSMs.
az cloudhsm list

# Show details of a Cloud HSM.
az cloudhsm show

# Update the properties of a Cloud HSM.
az cloudhsm update

# Begin a backup of the Cloud HSM.
az cloudhsm backup start

# Restore a backup of a Cloud HSM.
az cloudhsm restore start
```
