# az compute-fleet

```bash
# Create an Azure Compute Fleet.
az compute-fleet create

# Delete an Azure Compute  Fleet.
az compute-fleet delete

# List Fleet resources by subscription ID and resource group.
az compute-fleet list

# List VirtualMachineScaleSet resources by Fleet.
az compute-fleet list-vmss

# Get a Fleet.
az compute-fleet show

# Update a Fleet.
az compute-fleet update

# Place the CLI in a waiting state until a condition is met.
az compute-fleet wait
```
