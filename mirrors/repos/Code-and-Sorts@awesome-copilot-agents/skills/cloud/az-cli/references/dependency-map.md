# az dependency-map

```bash
# Create a MapsResource.
az dependency-map create

# Delete a MapsResource.
az dependency-map delete

# List MapsResource resources by subscription ID.
az dependency-map list

# Get a MapsResource.
az dependency-map show

# Update a MapsResource.
az dependency-map update

# Place the CLI in a waiting state until a condition is met.
az dependency-map wait

# Create a DiscoverySourceResource.
az dependency-map discovery-source create

# Delete a DiscoverySourceResource.
az dependency-map discovery-source delete

# List DiscoverySourceResource resources by MapsResource.
az dependency-map discovery-source list

# Get a DiscoverySourceResource.
az dependency-map discovery-source show

# Update a DiscoverySourceResource.
az dependency-map discovery-source update

# Place the CLI in a waiting state until a condition is met.
az dependency-map discovery-source wait
```
