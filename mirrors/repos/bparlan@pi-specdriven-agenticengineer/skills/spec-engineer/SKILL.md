# Spec-Engineer Skill

## Purpose
Turn vague ideas → numbered, verifiable tasks + full spec structure.

## How to invoke
/spec <your idea here>

## Examples

/spec Add user authentication with JWT and refresh tokens

→ produces full spec with tasks T1–T6

## Output format (fixed)
See the prompt template in ~/.pi/agent/prompts/spec.md
