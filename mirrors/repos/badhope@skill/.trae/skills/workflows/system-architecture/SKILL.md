# System Architecture

## Overview

Comprehensive system architecture design.

## Key Capabilities

- High-level architecture design
- Component decomposition
- Scalability planning
- Reliability and fault tolerance
- Data flow design
- Technology stack selection

## Architecture Patterns

- Microservices
- Event-driven
- Serverless
- Layered architecture
- CQRS
- Event sourcing

## Process Flow

1. **Decompose** - Break down system into components
2. **Design** - High-level architecture design
3. **Evaluate** - Evaluate architecture options
4. **Document** - Create architecture documentation
5. **Review** - Peer review and validation

## Output Artifacts

- Architecture design document
- System architecture diagrams
- Component interaction diagrams
- Data flow diagrams
- Deployment architecture
