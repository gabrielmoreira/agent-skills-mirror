# SEO Optimizer

## Overview

Search engine optimization for web content.

## Key Capabilities

- Keyword research and analysis
- Meta tag optimization
- Content optimization
- Structured data (Schema.org)
- Sitemap generation
- Performance optimization for SEO

## Process Flow

1. **Research** - Keyword research and competitor analysis
2. **Optimize** - Optimize content and meta tags
3. **Structure** - Add structured data markup
4. **Technical** - Technical SEO audit
5. **Report** - SEO analysis report

## Output Artifacts

- SEO audit report
- Keyword strategy document
- Optimized content
- Structured data markup
- Sitemap.xml
