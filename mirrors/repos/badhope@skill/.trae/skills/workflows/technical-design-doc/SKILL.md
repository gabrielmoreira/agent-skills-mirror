# Technical Design Document

## Overview

Create comprehensive technical design documents.

## Key Capabilities

- System architecture design
- API specification
- Database schema design
- Technical stack selection
- Security considerations
- Performance requirements

## Process Flow

1. **Architect** - Design system architecture
2. **Specify** - Define API specifications
3. **Schema** - Design database schema
4. **Document** - Write technical design doc
5. **Review** - Peer review and validate

## Output Artifacts

- Technical Design Document (TDD)
- System architecture diagrams
- API specification (OpenAPI/Swagger)
- Database schema (ER diagram)
- Sequence diagrams
