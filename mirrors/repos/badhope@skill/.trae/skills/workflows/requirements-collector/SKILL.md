# Requirements Collector

## Overview

Systematic requirements collection and validation skill.

## Key Capabilities

- User interviews and need gathering
- Requirement documentation (PRD, BRD)
- Stakeholder validation
- Requirement prioritization
- Acceptance criteria definition

## Process Flow

1. **Gather** - Collect user needs and pain points
2. **Document** - Create structured requirement documents
3. **Validate** - Confirm with stakeholders
4. **Prioritize** - Rank requirements by importance
5. **Refine** - Define acceptance criteria

## Output Artifacts

- Product Requirements Document (PRD)
- Business Requirements Document (BRD)
- User stories with acceptance criteria
- Requirement traceability matrix
