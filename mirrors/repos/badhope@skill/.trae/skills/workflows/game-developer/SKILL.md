# Game Developer

## Overview

Game development from concept to playable product.

## Key Capabilities

- Game concept and design
- 2D/3D game development
- Game mechanics implementation
- Asset generation (textures, sounds, music)
- Level design
- UI/UX for games
- Performance optimization

## Supported Engines

- **2D**: Phaser, Godot, Unity 2D, PixiJS
- **3D**: Unity, Unreal Engine, Godot 3D, Three.js
- **Web**: HTML5 Canvas, WebGL, Three.js
- **Mobile**: Unity Mobile, Godot Mobile

## Game Genres

- Platformers
- Puzzle games
- RPGs
- Strategy games
- Shooters
- Simulation games
- Casual games

## Process Flow

1. **Concept** - Game concept and design doc
2. **Prototype** - Quick playable prototype
3. **Develop** - Full game development
4. **Assets** - Generate art, sound, music
5. **Test** - Playtesting and bug fixes
6. **Polish** - Optimization and refinement

## Example Usage

```
"Create a 2D platformer game"
"Make a puzzle game with physics"
"Prototype a 3D first-person shooter"
"Generate assets for my game"
"Optimize my game for performance"
```
